pengenalan database mysql
1. mysql merupakan database yang paling banyak digunakan
2. paling mudah di install
3. paling mudah dipelajari
4. gratis
5. mysql termasuk kelompok database: RDBMS (Relationship Database Management System)
	database mana saja yang termasuk RDBMS:
	1. ORACLE
	2. SQL SERVER
	3. MYSQL
	4. POSTGRESQL
	5. SQLite
RDBMS merupakan database yang terdiri atas tabel-tabel
6. Selain kelompok RDBMS ada kelompok databse yang lain yaitu: No SQL (code politan)
	1. MongoDB
	2. Apache HBase
	3. Cassandra
	4. couchdb
